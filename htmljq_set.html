La parte mas importante de jquery es lograr la manipulacion del dom en HTML y es en esta parte donde jquery no spuede facilitar una gran cantidad de metodos relacionados con el DOM para el acceso y la manipulacion de elementos y atributos.

Que es el dom , es el data object model es el estandar para acceder a nuestro HTML.

Existen tres metodos para obtener el contenido desde el HTML Estos nos seran utilies para cuando queramos recuperar o Manipular la informacion del DOM. text() - Establece o devuelve el contenido de texto de los elementos seleccionados html() - Establece o devuelve el contenido de los elementos seleccionados (incluido el marcado HTML) val() - Establece o devuelve el valor de los campos del formulario.
